Discord でログイン Bot をサーバーに追加

主な機能

サーバー運営に必要なすべてを、ひとつのBotで

🎫

チケットシステム

パネルからワンクリックでサポートチケットを作成。スタッフロール管理や自動削除にも対応。

🏷️

リアクションロール

ボタンを押すだけでロールを取得・解除。トグル・排他・ワンアクションの3モードに対応。

🔊

VC 自動作成

トリガーチャンネルに参加するだけでプライベート VC を自動作成。退出後は自動削除。

📢

VC 募集

パネルから募集を投稿し、スレッドで参加者を集める。メンションロールで通知も可能。

📋

メンバーログ

メンバーの入退室を自動通知。招待リンクの追跡やカスタムメッセージにも対応。

⏰

Bump リマインダー

Disboard / Dissoku の Bump を自動検知し、2時間後にリマインダーを送信。

Web ダッシュボードで簡単管理

コマンドを覚えなくても、ブラウザからすべての設定を直感的に操作できます。
